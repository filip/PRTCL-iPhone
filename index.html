PRTCL-iPhone by filip

iPhone Application

Filip Visnjic 2010 // fvda.co.uk // CreativeApplications.Net Created at the V&A Decode openFrameworks Workshop in London 28/02/2010 iPhone 3Gs recommended Features: - Control particles and their motion by drawing on the screen. - Tilt control particles or use automated motion - Change colours in the settings panel or using one of the included presets in the menu below. - Download desktop application which you can control using your iPhone and OSC Application. Download from the AppStore here: http://itunes.apple.com/us/app/prtcl/id359060528?mt=8 Read about this here: http://www.creativeapplications.net/iphone/prtcl-iphone-openframeworks/ For feature requests add a task here: http://nokahuna.com/projects/10961/tasks?category=40934 Thanks to Joel Gethin Lewis, Zach Lieberman, Kyle McDonald, Arturo Castro, Andreas Muller, Eduard Prats Molner, Zach Gage, onedotzero and Victoria and Albert Museum.
